## How to Use

1. Open `image_prompts.csv` in the parent directory
2. Copy each prompt into Gemini, ChatGPT, or Midjourney
3. Generate the image
4. Save with the filename listed above
5. Refresh the HTML book page to see images appear

## Image Specifications

- **Format:** PNG preferred (or JPG)
- **Resolution:** At least 300 DPI for print
- **Size:** Cover should be 6x9 inches (1800x2700 pixels at 300 DPI)
- **Other images:** 5 inches wide minimum (1500 pixels at 300 DPI)
